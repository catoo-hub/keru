# Keru — Japanese Flashcards

Веб-приложение с флеш-карточками для изучения японского языка.

> **Первый поддерживаемый язык: Японский (日本語)**
> Интерфейс на русском языке. В будущем возможна поддержка других языков.

## Idea

Keru — платформа для изучения японского языка с помощью флеш-карточек. Пользователи могут учить слова, кандзи и фразы через интерактивные карточки с интервальным повторением для лучшего запоминания.

Поддерживаются два режима:

- **Русский → Японский** — видишь русское слово, вспоминаешь японский перевод (кандзи/кана)
- **Японский → Русский** — видишь японское слово/кандзи, вспоминаешь значение на русском или его написание

## Планируемые функции

- **Колоды карточек** — тематические наборы (повседневная жизнь, JLPT N5–N1, кандзи по уровням и т.п.)
- **Два направления** — Русский → Японский и Японский → Русский на выбор
- **Кандзи, слова, фразы** — изучение иероглифов, словарного запаса и выражений
- **Интервальное повторение (SRS)** — умное расписание: карточки появляются перед тем, как ты их забудешь
- **Отслеживание прогресса** — статистика изученных карточек, серии и слабые места
- **Аккаунты пользователей** — сохранение прогресса и синхронизация между устройствами
- **Свои колоды** — возможность создавать собственные наборы карточек

## Целевая аудитория

Русскоязычные пользователи, которые хотят изучать японский язык — от начинающих до продвинутых.

## Технологии

- **Backend:** Laravel (PHP)
- **Frontend:** Blade + Vite (JS/CSS)
- **БД:** MySQL / SQLite

## Дорожная карта

1. Базовый UI карточек (переворот, следующая/предыдущая)
2. Наполнение БД стартовыми колодами (Русский ↔ Японский)
3. Регистрация и авторизация пользователей
4. Алгоритм интервального повторения
5. Дашборд прогресса
6. Создание пользовательских колод
7. Поддержка дополнительных языков
